# Meta-Model

The meta-model is the global contract that makes the governance system executable and auditable as a whole.

It ties together:
- layers and ownership
- invariants
- relationships
- workflow
- artifact lifecycle
- decision framework
- security boundaries
- observability
- economics
- GitOps
- learning
- skills
