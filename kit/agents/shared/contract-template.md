# Sub-Agent Contract Template

## Mission

State the bounded responsibility of the sub-agent in one paragraph.

## Inputs

List the minimum input packet that the sub-agent requires.

## Outputs

List the required artifacts, summaries, and decisions that must be returned.

## Handoff

State who receives the output, which envelope to use, and what state must be updated.

## Stop Conditions

Describe the exact conditions that end the sub-agent execution.

## State Update

Describe which fields in the harness state must change after the sub-agent completes.

## Quality Rules

Describe the local quality bar, including determinism, atomicity, and risk-based checks.
