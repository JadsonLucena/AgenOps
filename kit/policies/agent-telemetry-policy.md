# Agent Telemetry Policy

## Scope

Define the operational signals used to understand how the kit is used over time.

## Requirements

Track operational signals that improve the kit over time.
- clarifying questions per task
- reruns per task
- cache hit ratio
- token budget usage
- test failure recovery time
- documentation update rate
- number of triggered artifacts per change
- unresolved findings per role

## Exceptions

- Deviations require explicit rationale and approval when they affect shared behavior, security, or compatibility.

## Evidence

- review notes
- validation output
- linked change record
