# Skills

Skills are reusable, modular capabilities that power AgenOps agents.

A skill is not a role. A role is an operational responsibility; a skill is a capability that can be reused across roles, workflows, and projects.

## Why skills exist
- reduce variability
- improve reuse
- standardize execution patterns
- make agent behavior more predictable
- allow capability growth without rewriting the whole system

## Core skill categories
- elicitation
- architecture
- documentation
- decision exploration
- validation
- security analysis
- review
- telemetry analysis
- economic analysis
- workflow orchestration
