# Supply-Chain Checklist

## Purpose

Confirm that the release inputs and outputs are traceable and reviewed.

## Items

- [ ] Dependency pinning is reviewed.
- [ ] SBOM is updated.
- [ ] Integrity checks passed.
- [ ] Provenance is recorded.
- [ ] License review is complete.
- [ ] High-risk dependency changes are approved.

## Pass condition

All items are checked or explicitly waived with approval.
