# Agent Execution Audit

## Scope

Record the important facts of each agent run.

## Requirements

Capture run id, role, phase, inputs, tools, decisions, findings, and final status.

- record the run id
- record the role
- record the phase
- record the inputs used
- record the tools used
- record the decisions taken
- record the findings
- record the final status

## Exceptions

- debug sessions may be less verbose, but they still need a run identifier

## Evidence

- decision summary
- execution log
- validation status
