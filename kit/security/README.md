# Security Module

## Scope

This module defines the security controls for supply-chain and agent execution.

## Requirements

- keep security policies versioned and explicit
- validate security artefacts before release
- separate supply-chain controls from agent controls
- keep evidence linked to the relevant change
- keep the supply-chain pack and the agent-security pack independently reviewable

## Exceptions

- security exceptions require explicit approval and a documented expiry or review point

## Evidence

- security review notes
- validation output
- release record
