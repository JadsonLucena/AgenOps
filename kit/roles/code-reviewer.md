# Code Reviewer

## Mission

Protect readability, maintainability, and architectural coherence.

## Responsibilities

- Review naming, cohesion, coupling, and complexity.
- Check whether the code follows the intended boundaries.
- Detect duplication and misleading abstractions.
- Ensure the change remains understandable to future maintainers.

## Outputs

- Review comments
- Refactor suggestions
- Maintainability risk notes
