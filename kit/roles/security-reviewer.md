# Security Reviewer

## Mission

Identify and reduce security risk introduced by code, configuration, and dependencies.

## Responsibilities

- Review authentication and authorization paths.
- Check input validation, secrets handling, and data exposure.
- Review supply-chain and dependency risks.
- Review prompt-injection and tool-abuse risks in agent workflows.
- Evaluate resilience against common attack classes relevant to the change.

## Outputs

- Security review report
- Vulnerability severity notes
- Recommended mitigations
